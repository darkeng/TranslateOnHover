# Security Policy

## Supported Versions

Currently, only the latest `master` branch and the most recent release of **Translate On Hover** are receiving security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a high priority for this project, especially since the extension handles cross-origin requests and API connections.

If you discover a security vulnerability within this project, please DO NOT open a public issue. Instead, please follow these steps:

1. Send an email or reach out to the repository owner directly. (If no email is listed, you can submit a draft security advisory through GitHub).
2. Detail the vulnerability: Include the steps to reproduce, the impact, and the potential exploit scenario.

**Response Time:** We will attempt to acknowledge the vulnerability within 48 hours and provide a timeline for the fix. 

Once the vulnerability is patched, a CVE or public disclosure may be made crediting your discovery.
